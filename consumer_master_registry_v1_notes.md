# Consumer Ontology V1 Notes

This v1 set expands the earlier seed using recurring concepts observed across:
- Coca-Cola
- Nike
- PepsiCo
- Target
- Mondelez

What changed from seed to v1:
- stronger CPG coverage from Mondelez
- more explicit channel and retail operating concepts
- broader driver vocabulary for commercial, cost, macro, and execution themes
- more dimension types for customer, route-to-market, and alternate reporting basis

Recommended use:
- keep this as the master internal ontology candidate set
- do shortlist retrieval before sending candidates to the model
- continue separating canonical metrics, dimensions, and drivers instead of flattening them into one list

Most repeated cross-company concepts:
- pricing, volume, mix
- brand, category, segment
- inventory and digital/omnichannel themes
- inflation, commodity cost, and foreign currency drivers
